    UJI Smart Campus Web

    UJI Smart Campus Web

    Descripción del proyecto

    La web desarrollada para el Smart Campus de la Universitat Jaume I (UJI) es el portal web del Smart Campus de la UJI: http://smart.uji.es

    Es el portal web de la plataforma UJI SmartCampus. Esta plataforma consta de un gis con toda la información del campus; servicios para acceder a estos datos; servicios para acceder a la base de datos institucional y otros servicios funcionales. Este portal no es solo un visor GIS, ya que además del descubrimiento de información tiene diversas funcionalidades útiles para ayudar a los usuarios del campus. La funcionalidad GIS desarrollada en el portal se ha implementado mendiante la API de JavaScript de ArcGIS.

    Para esta aplicación se desarrollaron servicios de MapService, FeatureService y Network Analysis Service con ArcGIS Server, para los mapas y para el cálculo de rutas.

    Entre sus funcionalidades a destacar encontramos:

    • Búsquedas: por despacho, por personal, por departamento, por servicio,...
    • Rutas en interiores y exteriores
    • Visualización consumos energéticos
    • Visualización en 3D

    UJI Smart Campus App

    UJI Smart Campus App

    Project Description

    UJI Smart Campus App, es la app nativa de Android de la plataforma de Smart Campus de la UJI. Esta app está diseñada para que sea una herramienta útil tanto para los estudiantes de la Universitat Jaume I como para el personal.

    Entre sus funcionalidades destacan:

    • Navegación en el mapa por el interior de los edificios
    • Descubrimiento en el mapa de Oficinas, despachos, aulas, servicios universitarios,...
    • Buscador: despachos, aulas, personal universitario,...
    • Rutas en exteriores e interiores
    • Sistema de posicionamiento en Interiores y exteriores

    Para su implementación se ha utilizado el SDK de ArcGIS para Android.

    Bicicas-Renfe Trip Planner

    Bicicas-Renfe Trip Planner

    Project Description

    Este proyecto consite en la integración del servicio de préstamo de bicicletas de Castelló de la Plana y el servicio de trenes de Renfe.

    Para su desarrollo se ha utilizado el API de JavaScript de ESRI. La aplicación hace lo siguiente:

    1. Calcula la ruta más corta hasta la estación de bicicletas más cercana con bicicletas disponibles (teniendo en cuenta que después se quiere ir a la estación de tren)
    2. Calcula la ruta hasta la estación de tren
    3. Muestra los horarios de trenes a partir del tiempo que se tarda en llegar a la estación.

    Esta web app ha sido premiada en el concurso Geo Developers Code Challenge al Mejor Proyecto Social.

    Bus Castellón

    Bus Castellón

    Project Description

    Esta aplicación muestra a tiempo real el horario de los autobuses públicos de Castellón. Pulsando sobre cualquier parada, muestra abajo la información de los dos próximos autobuses a esa parada. El tiempo no es el horario de los autobuses, sino que se conecta a un servicio que ofrece el tiempo de espera a tiempo real.

    Esta app ha sido desarrollada para Android, usando el API de Google Maps.

    App Development
    Parking Now

    Parking Now

    Project Description

    Parking Now es una plataforma de colaboración ciudadana, en la que se pretende que los ciudadanos situen en el mapa plazas de aparcamiento en las zonas dónde habitualmente aprarquen. La aplicación es capaz de detectar que se ha aparcado en una plaza (previamente mapeada) cambiando el estado de la misma. La app también te calcula la ruta hasta el punto dónde quieras aparcar, haciendo un análisis sobre la disponibilidad de plazas cerca del destino seleccionado.

    Otra funcionalidad que tiene, es que una vez aparcado el coche, puedes buscarlo gracias a la realidad aumentada.

    Esta app fue ganadora del Premio Especial Lunacloud en el Geo Developers Code Challenge.
