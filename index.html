Web development
UJI Smart Campus Web

UJI Smart Campus Web

Descripción del proyecto

La web desarrollada para el Smart Campus de la Universitat Jaume I (UJI) es el portal web del Smart Campus de la UJI: http://smart.uji.es

Es el portal web de la plataforma UJI SmartCampus. Esta plataforma consta de un gis con toda la información del campus; servicios para acceder a estos datos; servicios para acceder a la base de datos institucional y otros servicios funcionales. Este portal no es solo un visor GIS, ya que además del descubrimiento de información tiene diversas funcionalidades útiles para ayudar a los usuarios del campus. La funcionalidad GIS desarrollada en el portal se ha implementado mendiante la API de JavaScript de ArcGIS.

Para esta aplicación se desarrollaron servicios de MapService, FeatureService y Network Analysis Service con ArcGIS Server, para los mapas y para el cálculo de rutas.

Entre sus funcionalidades a destacar encontramos:

  • Búsquedas: por despacho, por personal, por departamento, por servicio,...
  • Rutas en interiores y exteriores
  • Visualización consumos energéticos
  • Visualización en 3D
App Development
UJI Smart Campus App

UJI Smart Campus App

Descripción del proyecto

UJI Smart Campus App, es la app nativa de Android de la plataforma de Smart Campus de la UJI. Esta app está diseñada para que sea una herramienta útil tanto para los estudiantes de la Universitat Jaume I como para el personal.

Entre sus funcionalidades destacan:

  • Navegación en el mapa por el interior de los edificios
  • Descubrimiento en el mapa de Oficinas, despachos, aulas, servicios universitarios,...
  • Buscador: despachos, aulas, personal universitario,...
  • Rutas en exteriores e interiores
  • Sistema de posicionamiento en Interiores y exteriores

Para su implementación se ha utilizado el SDK de ArcGIS para Android.
