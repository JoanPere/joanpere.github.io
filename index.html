    UJI Smart Campus Web

    Descripción del proyecto

    La web desarrollada para el Smart Campus de la Universitat Jaume I (UJI) es el portal web del Smart Campus de la UJI: http://smart.uji.es

    Bicicas-Renfe Trip Planner

    Bicicas-Renfe Trip Planner

    Project Description

    Este proyecto consite en la integración del servicio de préstamo de bicicletas de Castelló de la Plana y el servicio de trenes de Renfe.

    Para su desarrollo se ha utilizado el API de JavaScript de ESRI. La aplicación hace lo siguiente:

    LiDAR
    Detección de dolinas con LiDAR

    Detección de dolinas con LiDAR

    Project Description

    El objeto de este proyecto fue la identificación de dolinas (sumideros) en la zona de Oskotz. La delimitación de estas dolinas se realizó utilizando la técnica del vuelo LiDAR. Esta tecnología permite la obtención de puntos en el suelo incluso en zonas boscosas. Se necesitaba ubicar estas dolinas, conocer el número que había y el tamaño de estas ya que para delimitar de forma precisa la cuenca hidrográfica se deben de conocer todos los cursos que sigue el agua pluvial.

    Para este proyecto se realizó la planificación del vuelo para la suficiente densidad de puntos en el terreno; el cálculo de la trayectoria del avión para la georeferenciación de la nube de puntos LiDAR; el procesado y clasificación de los puntos; y por último la obtención de las dolinas mediante un análisis SIG del MDT generado a partir de la nube de puntos.

    Parking Now

    Parking Now

    Project Description

    Parking Now es una plataforma de colaboración ciudadana, en la que se pretende que los ciudadanos situen en el mapa plazas de aparcamiento en las zonas dónde habitualmente aprarquen. La aplicación es capaz de detectar que se ha aparcado en una plaza (previamente mapeada) cambiando el estado de la misma. La app también te calcula la ruta hasta el punto dónde quieras aparcar, haciendo un análisis sobre la disponibilidad de plazas cerca del destino seleccionado.

    Otra funcionalidad que tiene, es que una vez aparcado el coche, puedes buscarlo gracias a la realidad aumentada.

    Esta app fue ganadora del Premio Especial Lunacloud en el Geo Developers Code Challenge.

    Cubicación Salinas con LiDAR

    Cubicación Salinas con LiDAR

    Project Description

    El objeto de este trabajo fue el cálculo del movimiento de tierras realizado en la salina de Beriain entre los años 2008 y 2009. Para ello se generó un Modelo Digital del Terreno (MDT) del año 2009 de la zona de trabajo con tecnología LiDAR y se compararó con los modelos de fechas anteriores generados a partir de la información facilitada de años anteriores..

    EL proyecto consitió en la realización del vuelo LiDAR, el procesado de los datos LiDAR para obtener el MDT, y la realización de un perfil longitudinal y los diferentes perfiles transversales.

    UJI Parking Real Time App

    UJI Parking Real Time App

    Project Description

    Esta App permite ver el estado actual (libre, ocupada, solicitada,...) de las plazas de parking de la Universitat Jaume I.

    Project Description

    La Fundación Fundación Luis Giménez Lorente de cartografía histórica de la Universidad Politécnica de Valencia, cuenta unos fondos cartográficos de capital importancia para la comunidad Universitaria y el público en general del siglo XVI-XVIII.
